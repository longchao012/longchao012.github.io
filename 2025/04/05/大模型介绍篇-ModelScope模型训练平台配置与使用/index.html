详细介绍ModelScope魔搭社区平台的使用方法，包括模型下载、推理、微调和部署的完整流程，助力AI开发者高效使用开源模型。

一、快速开始

先熟悉魔搭社区的模型下载和模型推理。设置开发环境，并顺利从魔搭社区开始模型学习，使用和开发。

二、环境安装

可以使用魔搭社区已经预安装好的 Notebook 环境来使用魔搭社区模型。魔搭社区 Notebook 为新用户提供了 100小时的免费 GPU 算力和不限时长的免费 CPU 算力，并预安装了大部分模型可运行的环境依赖。

六、模型部署

您可以使用魔搭社区 SwingDeploy 来一键部署指定的模型到云资源上。SwingDeploy 除了支持各种任务小模型的部署外，对于部分大语言模型，SwingDeploy 支持部署后直接提供 OpenAI API 兼容的调用接口。
